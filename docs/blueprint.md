# **App Name**: WaveSync ERP

## Core Features:

- Financial Overview Dashboard: Displays key financial metrics including revenue, expenses, and profit through interactive charts and summaries.
- Broadcast & Appointment Calendar: An interactive calendar interface for scheduling broadcasts, managing guest appearances, and organizing team appointments.
- Client & Guest Directory: Comprehensive management system for advertiser and guest profiles, including contact details, interaction history, and linked campaigns.
- Ad Campaign Manager: Tool for tracking, allocating, and managing advertising slots and campaigns, including campaign duration and status updates.
- AI-Powered Scheduling Assistant: Utilizes a tool to analyze historical broadcast data and station guidelines to suggest optimal scheduling for programs and ad placements.
- Performance Reporting: Generates detailed reports on financial performance, audience engagement, and ad campaign effectiveness over specified timeframes.

## Style Guidelines:

- Background: Very dark blue-grey (#161C1E) to establish a sophisticated and calming dark theme, reflecting a professional interface.
- Primary elements: Muted dark blue (#336699) for interactive components, subtly enhancing readability and functionality on the dark background.
- Accent color (blue): Vibrant blue (#4293F0) for active states, important data visualization, and key indicators, evoking professionalism and technology, per user request.
- Accent color (orange): Energetic orange (#F57E1A) for call-to-action buttons, warnings, and highlighting crucial information, providing dynamic contrast, per user request.
- All text: 'Inter' (sans-serif) for its modern, clear, and objective readability, ideal for data-heavy displays and professional content.
- Line-based, modern, and minimalistic icons, providing clarity and quick visual understanding within the professional dark theme.
- A clean, modular layout with clear information hierarchy and consistent spacing, optimizing for data display and efficient navigation.
- Subtle and purposeful micro-interactions and transitions to provide feedback and enhance the fluidity of the user experience, particularly in calendar interactions and chart updates.
- Light cream background (#FFF3E0) with dark orange text (#B36000).
- Light green background (#E8F5E9) with dark green text (#1B6B35).
- Light purple background (#F3E5F5) with deep purple text (#7B1FA2).
- Light pink background (#FCE4EC) with dark pink text (#AD1457).
- Uses CSS variables for background `(--bg3)` and text `(--ink3)`.
- Uses CSS variables for light red `(--red-light)` and red text `(--red)`.
